## 專案名稱：JAVA-簡易版世紀帝國(三種驗算法路徑)
建立一個簡易版本的世紀帝國，包含地圖、障礙物、士兵、農民及生產單位，所有可移動單位有三種不同的移動路徑演算法 (BFS、DFS 及 UCS)

---
## 專案目的
利用 JAVA 建立一個簡易的世紀帝國，並包含以下特點：
1. 需要一個包含障礙物的地圖，障礙物上不可建造建築物、不可移動、可移動單位不可穿越、障礙物不僅有1X1大小
2. 生產單位可生產農民，需有數字比例提示當前建造進度及生產進度
3. 生產後的農民於一定秒數以後變更為士兵，變更途中需轉換成對應階段的圖片
4. 士兵及農民有三種不同的移動演算法：
   - BFS 廣度搜尋法
   - DFS 深度搜尋法
   - UCS 最短路徑搜尋法
